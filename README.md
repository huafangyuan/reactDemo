# react demo
react学习示例

### intall
* Just clone the repo and install the necessary node modules

```
$ git clone git@github.com:just-me-cy/reactDemo.git
$ cd reactDemo
$ npm i 
$ npm start
```
* open your browser http://localhost:8080/build/
